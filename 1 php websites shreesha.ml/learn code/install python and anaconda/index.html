Install Jupyter Notebook on Ubuntu

Upgrade and Update

sudo apt-get update
sudo apt-get upgrade

Install Jupyter Notebook without Anaconda

Install Python3, iPython, Pip3

sudo apt-get install python3
sudo apt-get install ipython3
sudo apt-get -y install python3-pip
sudo apt-get update

first go into root using the command 

sudo su

install notebook

pip3 install jupyter
exit
jupyter notebook – allow-root

To create a new directory using the terminal use this command

mkdir -p anaconda

Download Anaconda using Browser

Click here for downloading website

By default, the file will be downloaded to the Downloads folder. You will need to right-click the downloaded .sh file and then click on “move”. Then select the new anaconda folder that you created and move the downloaded .sh file to that folder.


To get to Download Anaconda using Terminal click here

Install Jupyter Notebook using Anaconda

conda install -c conda-forge notebook

After Jupyter Notebook finishes installing, you can run it using the command

jupyter notebook
